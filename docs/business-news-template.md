# 업체별 소식 입력 템플릿

이 파일은 `town-map` 업체 상세의 `소식` 탭 데이터를 채우기 위한 입력 템플릿입니다.

## 필드 설명

`title`
- 소식 카드 상단 제목입니다.
- 한 줄 텍스트로 작성합니다.

`body`
- 소식 본문입니다.
- 2~4문장 정도로 자연스럽게 작성합니다.
- 줄바꿈이 있으면 그대로 전달해도 됩니다.

`created_at`
- 소식이 올라온 시각입니다.
- 화면에서는 이 값을 기준으로 `몇 시간 전`, `며칠 전`처럼 표시됩니다.
- 형식: `2026-04-14T09:00:00+09:00`
